ASCII 코드란?

American Standard Code for Information Interchange의 약자로, 영문 알파벳과 숫자, 특수문자를 0~127 사이의 숫자로 표현하는 방식입니다.

  • 총 128개의 문자 표현 가능
  • 영어와 기본 기호만 지원
  • 1바이트(8비트) 사용
🌐

유니코드란?

전 세계의 모든 문자를 컴퓨터에서 일관되게 표현하고 다룰 수 있도록 만든 국제 표준 문자 코드입니다.

  • 전 세계 모든 언어 지원
  • 이모지도 포함
  • 1~4바이트 가변 길이
